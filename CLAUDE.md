# IELTS 备考仓库

三个月目标 7.0 分自学方案

## 目录结构

| 目录 | 内容 |
|------|------|
| `01_听力` | 王陆语料库、剑雅真题、复盘笔记 |
| `02_阅读` | 词汇、538考点词、真题 |
| `03_写作` | Simon/杜仕明框架、范文、复盘 |

## 核心规则

- 听力：每天一套剑雅 + 王陆语料库 3个test
- 阅读：平行阅读法 + 538考点词 + 不背单词30min/天
- 写作：考前2周每天复盘框架
- 口语：每天1h真题练习
- 剑雅顺序：7 → 14 → 15 → (9,10,11) → 16 → 17 → 18 → 19 → 20
- 做题节奏：2天一套（第一天S1+S2+P1+P2，第二天S3+S4+P3）
